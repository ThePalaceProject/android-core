org.librarysimplified.viewer.pdf.pdfjs
===

The `org.librarysimplified.viewer.pdf.pdfjs` module is an implementation
of the [Viewer SPI](../palace-viewer-spi/README.md) that uses 
pdf.js to display PDF files.
