org.librarysimplified.opds2.pwp
===

The `org.librarysimplified.opds2.pwp` module provides a parser implementation for OPDS 2.0 feeds
that uses the Palace WebPub OPDS parser internally.

#### See Also

* [Palace WebPub](https://github.com/ThePalaceProject/android-webpub)
* [OPDS 2.0](https://drafts.opds.io/opds-2.0)
